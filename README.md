# Banlieusards — site statique gratuit

Version épurée fond blanc.

## Voir le site en local
1. Dézipper le dossier.
2. Ouvrir `index.html` dans Safari ou Chrome.

## Modifier le domaine
Le fichier `CNAME` contient le domaine utilisé par GitHub Pages.
Vérifie l’orthographe : `balieusards.store` ou `banlieusards.store`.
